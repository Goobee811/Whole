# Editing Workflow

## Standard Edit Flow

### 1. Preparation
- Activate `whole-analyzer`
- Generate section analysis
- Review findings

### 2. Planning
- Identify changes needed
- Plan cross-ref updates
- Draft change proposal

### 3. Approval
- Present proposal to user
- Wait for explicit approval
- Clarify any questions

### 4. Implementation
- Activate `whole-editor`
- Apply changes systematically
- Update cross-refs bidirectionally

### 5. Validation
- Activate `whole-reviewer`
- Run full validation
- Fix any issues

### 6. Completion
- Generate summary report
- Document changes made
- Suggest follow-up actions

## Quality Gates

Gate 1: Analysis complete before proposing
Gate 2: Explicit approval before implementing
Gate 3: Validation pass before completion
