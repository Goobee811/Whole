# Bilingual Content Rules for Whole

## Vietnamese Primary Language

### Terminology Standards
- Use established Vietnamese academic terminology
- Respect cultural context and philosophical traditions
- Maintain consistency across all documents

### Examples
- ✅ "Tư duy hệ thống" (Systems Thinking)
- ✅ "Nhận thức meta" (Meta-cognition)
- ❌ "Hệ thống suy nghĩ" (too literal)

## English Secondary Language

### Translation Approach
- **Conceptual precision** over literal translation
- Accessible to international readers
- Preserve technical accuracy

### Format Requirements
```markdown
## Concept Name | Tên Khái Niệm

**Definition | Định nghĩa**
English text | Vietnamese text

**Context | Ngữ cảnh**
English text | Vietnamese text

**Application | Ứng dụng**
English text | Vietnamese text

**Integration | Tích hợp**
English text | Vietnamese text
```

## Cultural Authenticity

### Vietnamese Context
- Reference Vietnamese philosophers when appropriate
- Use culturally relevant examples
- Respect Buddhist/Confucian influences where applicable

### Universal Accessibility
- English should be understandable globally
- Avoid Vietnam-specific references in English without explanation
- Use universal examples when possible

## Translation Quality Standards

### Vietnamese → English
- Preserve conceptual depth
- Maintain academic rigor
- Use internationally recognized terminology where applicable
- Clarify culture-specific concepts

### Consistency Requirements
- Same concept = same translation across all documents
- Maintain terminology database
- Regular review of translation choices
- Document rationale for non-literal translations

## Common Translation Patterns

| Vietnamese | English | Notes |
|------------|---------|-------|
| Nhận thức | Cognition/Awareness | Context-dependent |
| Tư duy | Thinking/Thought | Preserve nuance |
| Hệ thống | System/Systematic | Check adjective/noun |
| Tích hợp | Integration | Standard term |
| Phản hồi | Feedback | Technical context |

## Review Process

1. **Self-review**: Check alignment after writing
2. **Peer review**: Cross-check with existing translations
3. **Consistency check**: Verify terminology database
4. **Cultural validation**: Ensure authenticity preserved
