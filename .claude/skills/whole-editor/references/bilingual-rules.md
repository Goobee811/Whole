# Bilingual Content Rules

## Vietnamese Primary Language

### Standards
- Use established Vietnamese academic terminology
- Respect cultural context and philosophical traditions
- Maintain consistency across all documents

### Good Examples
- ✅ "Tư duy hệ thống" (Systems Thinking)
- ✅ "Nhận thức meta" (Meta-cognition)
- ❌ "Hệ thống suy nghĩ" (too literal)

## English Secondary Language

### Translation Approach
- **Conceptual precision** over literal translation
- Accessible to international readers
- Preserve technical accuracy

## Format Template

### Tên khái niệm (chuẩn duy nhất)
```
#### **[số]. Tên English - Tên Tiếng Việt**
```

### Nội dung khái niệm
```markdown
#### **[số]. Tên English - Tên Tiếng Việt**

[Mô tả ngắn về khái niệm bằng tiếng Việt]

- **[English term]**: [Vietnamese explanation]
- **[English term]**: [Vietnamese explanation]
- **[English term]**: [Vietnamese explanation]
- **[English term]**: [Vietnamese explanation]
- ... (tối thiểu 4 points, có thể nhiều hơn)

→ **Liên kết:** [Cross-references đến domain/concept liên quan]
```

## Cultural Considerations

### Vietnamese Context
- Reference Vietnamese philosophers when appropriate
- Use culturally relevant examples
- Respect Buddhist/Confucian influences

### Universal Accessibility
- English understandable globally
- Explain Vietnam-specific references
